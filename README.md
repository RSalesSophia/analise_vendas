#### Análise de dados para uma companhia do setor de vestuário

Esse repositório possui dois scripts escritos no jupyter notebook em Python e um dashboard construído no Power BI. 
Sendo o primeiro script referente ao processo de ETL realizado nas bases de dados, o segundo correspondente a análise descritiva dos dados especificamente para *produtos* e por fim um dashboard que possui insights e indicadores relativo a vendas e clientes da empresa.
